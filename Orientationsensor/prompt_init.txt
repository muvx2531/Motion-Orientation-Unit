status_flags bit definition:

* bit0 = MPU6500 OK
* bit1 = LIS3MDL OK
* bit2 = DPS310 OK
* bit3 = GNSS OK
* bit4 = MPU6500 new data
* bit5 = LIS3MDL new data
* bit6 = DPS310 new data
* bit7 = GNSS new data
* bit8 = magnetic data may be disturbed
* bit9 = pressure reference set
* bit10 = calibration data loaded

Do not send very long JSON strings as the default format.
CSV is preferred for the first firmware version because it is easy to inspect using a serial terminal, Python, Excel, or logging tools.
